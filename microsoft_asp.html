ASP.NET is a unified web development model integrated with .NET framework, designed to provide services to create dynamic web applications and web services. It is built on the Common Language Runtime (CLR) of the .NET framework and includes those benefits like multi-language interoperability, type safety, Memory collection and inheritance. ASP.NET facilitate the development of distributed applications in structured and object-oriented manner by separating the presentation and content and hence write clean code. ASP.NET uses the code-behind model to generate dynamic pages MVP, MVC architecture. They have some the major differences from ASP, an earlier version of ASP.NET. The object model of ASP.NET has thus significantly improved from ASP, which makes it fully backward compatible to ASP.

  • Usage of compiled code (instead of interpreted code)
  • Event-driven server-side scripting model(Server Control, User Control)
  • State management – Session, Applicatio, Cookies, Context
  • Rapid application development using controls and libraries of the .NET framework.
  • Capacity provided by ASP.NET development framework.
  • Dynamic programming code is placed separately in a file or specially designated tag. This avoids the program code getting modified during runtime
